# MARK.ART16 — Maquette web immersive

Prototype de présentation client pour un réalisateur / photographe.

## Ouvrir la maquette
Double-cliquer sur `index.html`.

## Ce qui est déjà intégré
- Introduction plein écran avec bouton `ENTER THE FRAME`
- Univers noir et blanc premium avec placeholders roses `[PHOTO]` / `[VIDEO]`
- Hero vidéo fictive animé au scroll
- Projets sélectionnés avec ouverture d'une page-projet simulée
- Galerie type feed artistique
- Section services et formulaire de demande de projet
- Curseur personnalisé, grain, flash et transitions
- Version responsive mobile

## Fichiers à remplacer plus tard
Les blocs roses pourront être remplacés par les vraies photos et vidéos de Mark.
